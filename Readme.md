# ATM-Task

This is a practice task covering all python basics.
It includes
  - Reading/Writing to CSV
  - Using Pandas
  - Basic I/O from CLI

  - Send money to other accounts


You can also:
  - Create your account
  - Login
  - Withdraw or Deposit money

This requires Python 3.7 to run.
Install the following dependencies.
 - Pandas
